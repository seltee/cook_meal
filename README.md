# 🍳 Fridge-to-Meal Flutter App

A work-in-progress Flutter application that helps you decide what to cook based on the ingredients you already have at home.

The app allows users to store ingredients in a virtual "fridge" using Firebase, browse a list of meals, and match available ingredients with possible recipes.

---

## Live Demo

[Open the web app](https://seltee.github.io/cook_meal/)

## Features

### Currently Working

- **Firebase Authentication**
  - User sign-in and authentication are fully integrated.

- **Fridge Management**
  - Add ingredients to your fridge.
  - Remove ingredients when used or expired.
  - Data is stored in Firebase.

- **Meal List**
  - Browse a list of available meals.

- **Dashboard**
  - Basic overview of your data and functionality.

- **Dark / Light Theme**
  - Toggle between dark and light modes.
